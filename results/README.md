# Results

Store generated experiment outputs here.

This directory is scaffold-only until the debug and proposal-scale runs produce
metric JSON files, plots, or short summaries worth committing.

Examples:

- layerwise hidden-state probe metrics in JSON
- Markdown tables produced by `token-vs-context summarize`
- plots of `R^2`, cosine similarity, and MSE by layer
- short experiment summaries comparing models, dataset slices, or probe settings
- later, feature-level SAE predictability summaries once that extension is implemented
